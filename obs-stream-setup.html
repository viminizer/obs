Stream kit

OBS setup, automated.

One script installs your plugins, builds an NVENC streaming profile, and drops in five ready-made scenes — with animated overlays, live CS2 stats, replay clips, audio ducking, and auto scene switching wired in. This page shows how to run it and what you get.

Target: Windows + NVIDIA Output: 1080p60 · 9000 kbps Encoder: NVENC H.264 Streaming via: Restream → YouTube Overlay theme: acid · printstream · purple
Run it on the streaming PC
1

Copy the whole folder

Clone the repo (or copy the entire folder) to the Windows PC — the overlays\ and gsi\ folders must come along, the script installs from them.

2

Double-click run-setup.bat

It asks for admin once (UAC) and installs silently. Default overlay theme is acid; pass another to switch.

# default (acid) — or pick a theme:
run-setup.bat
run-setup.bat printstream
run-setup.bat purple
3

Pick devices in OBS

Open OBS → choose the Stream 1080p60 NVENC profile and Stream Kit collection → click each capture/webcam/mic source once to select your hardware.

Stream automation — wired in by the script
FeatureHow it behaves
Live CS2 statsCS2 pushes game state (official Valve GSI) to a tiny local server; the Gaming scene shows HP / armor / money / K-D-A / round score + a BOMB PLANTED banner. Start %APPDATA%\obs-studio\gsi\run-gsi.bat before streaming — without it the bar simply hides.
Dynamic HUD chipsThe bottom ticker's MAP chip shows the live map name from the game. The RANK chip reads GLOBAL; put your Premier rating in gsi\premier-elo.txt (just the number) and it switches to PREMIER <rating> — editable mid-stream, never overwritten by re-runs.
Auto scene switchFocusing the CS2 window switches OBS to the Gaming scene (built-in switcher, pre-configured).
Audio duckingDesktop Audio has a compressor sidechained to the mic — game volume dips while you talk.
Replay clipsF10 saves the last 60 s to your Videos folder. Buffer starts with the stream automatically.
Animated overlaysWebcam frame, HUD ticker, animated background, countdown + BRB screens — all browser sources, theme chosen at install (acid default). Edit text in %APPDATA%\obs-studio\overlays\*.html, then right-click the source → Refresh.
Fast re-runsrun-setup.bat skips plugins that are already installed (no re-downloads), but rebuilds profile + scenes. For overlay tweaks only, use update-overlays.bat — copies just the HTML files, keeps your device picks and OBS tweaks intact.
Before you go live

Your manual steps

  • Pick your devices once — Display Capture, Webcam, Mic, Desktop Audio (step 3 above).
  • Idle → BRB rule (one time, ~30 s): Tools → Advanced Scene Switcher → Idle tab → enable, switch to BRB after 120 seconds. The script can't pre-build this one safely.
  • Before each stream: double-click %APPDATA%\obs-studio\gsi\run-gsi.bat and leave the window open — it powers the live CS2 stats.
  • Settings → Stream: confirm Restream/YouTube is still connected.
  • Optional — Premier rating on the HUD: put the number in %APPDATA%\obs-studio\gsi\premier-elo.txt. Leave 0 and the chip shows RANK GLOBAL instead.
  • Nothing is destructive — the script only adds a new profile and scene collection. Your existing ones stay untouched. Re-run it any time (e.g. to switch themes).
