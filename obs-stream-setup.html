Stream kit

OBS setup, automated.

One PowerShell script installs your plugins, builds an NVENC streaming profile, and drops in five ready-made scenes. This page shows how to run it — and how each scene looks on stream.

Target: Windows + NVIDIA Output: 1080p60 · 6000 kbps Encoder: NVENC H.264 Streaming via: Restream
Run it on the streaming PC
1

Copy the script over

Move obs-stream-setup.ps1 to the Windows PC (USB, cloud, or Downloads). The HTML can ride along.

2

Run in PowerShell

Open the folder, then run the command. It asks for admin once (UAC) and installs silently.

# in the script's folder
powershell -ExecutionPolicy Bypass -File .\obs-stream-setup.ps1
3

Pick devices in OBS

Open OBS → choose the Stream 1080p60 NVENC profile and Stream Kit collection → click each capture/webcam/mic source once to select your hardware.

Before you go live

Two things to know

  • Gaming and Coding now both capture the whole screen (same Display Capture source). Keep both as labels for scene-switching, or delete one if you don't need it.
  • StreamFX was dropped on purpose. It's abandoned (last real build 2023) and breaks on OBS 31. Move transition covers what most people used it for.
  • Nothing is destructive — the script only adds a new profile and scene collection. Your existing ones stay untouched.
