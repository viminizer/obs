Adding a Browser Source in OBS: Sources panel → + → Browser → check Local file → Browse to the .html → set Width/Height → OK. Tick "Refresh browser when scene becomes active" so animations restart cleanly.

To edit text (handle, socials, map name): open the file in any editor and change the words near ZEVON / @zevon / DE_MIRAGE. Save, then right-click the source → Refresh.
